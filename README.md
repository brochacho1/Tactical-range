# Tactical Range Simulator

## How to Play (The Easy Way)
You don't need to download anything! 
1. Open the **Shared App URL** in Chrome or Edge.
2. Click the **Install** icon in the address bar.
3. A shortcut will appear on your **Desktop**. Double-click it to play!

## How to Run Locally (For Developers)
If you want to run the code on your own computer:
1. Install [Node.js](https://nodejs.org/).
2. Extract this ZIP folder.
3. Double-click `start_game.bat`.
4. Open `http://localhost:3000` in your browser.
